# MAI SPA

Лендинг премиального тайского SPA-салона **MAI SPA** (Йошкар-Ола).

Самодостаточный статический сайт с фирменными скролл-эффектами:
частицы **собираются в единое целое** (лотос → логотип «MAI SPA») и рассыпаются
обратно при скролле, кастомный курсор, magnetic-кнопки, reveal-анимации,
parallax, tilt-карточки и count-up статистика.

## Структура

```
index.html            # разметка страницы
assets/css/style.css  # дизайн-система и стили
assets/js/particles.js# canvas-движок частиц (сборка/рассыпание формы)
assets/js/app.js      # оркестрация UI и скролл-эффектов
design.md             # бриф дизайн-языка (по подходу design.md)
```

## Запуск

Никакой сборки не нужно. Любой статический сервер:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

Или просто открыть `index.html` в браузере.

## Технологии

- Vanilla JS, Canvas 2D — без внешних runtime-зависимостей
- Шрифты: Cormorant Garamond + Manrope (с системным fallback)
- Адаптив mobile-first, поддержка `prefers-reduced-motion`

## Что заменить перед публикацией

- Реальные фотографии в слоты `data-photo` (галерея, карточка «Философия»)
- Телефон, адрес и ссылки соцсетей в секции «Контакты» (`index.html`)
- Цены и длительность программ в секции «Услуги» при необходимости
- Подключить отправку формы заявки на бэкенд/почту
