# MAI SPA

Иммерсивный 3D-сайт премиального тайского SPA-салона **MAI SPA** (Йошкар-Ола).

Весь сайт — одна WebGL-сцена: скролл управляет полётом камеры по
Catmull-Rom-кривой через 5 пространств (тоннель → зал ритуалов →
хаммам → зал мастеров → лотос/контакты). Чистый WebGL без зависимостей.

## Что внутри

- `index.html` — иммерсивный сайт (вся вёрстка, стили и движок в одном файле)
- `classic.html` + `assets/` — прежняя 2D-версия с частицами (сохранена)
- `assets/img/` — фото (мастера, массаж)
- `.github/workflows/pages.yml` — деплой на GitHub Pages

## Фичи

- Полёт камеры по кривой, 5 пространств-«комнат», параллакс за курсором/гироскопом
- Экран-приглашение «Войти в MAI SPA»
- Генеративный звук (WebAudio, без файлов): тёплый дрон, вода, один гонг
  на входе, затем тихая тайская релакс-мелодия (пентатоника); кнопка mute
  (состояние запоминается)
- Полёт «прилипает» к комнатам: возле точки скорость гаснет, после остановки
  камера мягко дотягивается к ближайшей комнате (сильный скролл — пропускает)
- Интерактивные карточки услуг: hover-подсветка, клик — подлёт камеры
  и панель с описанием + «Записаться» (yclients)
- Финал: частицы собираются в лотос и перетекают в надпись MAI SPA
- Портреты мастеров в светящихся порталах
- Прогресс-рейл: 5 точек, клик — плавный перелёт
- Фолбэк: плоская версия при reduced-motion / без WebGL (`?flat=1`)
- Весь текст в DOM (SEO/доступность)

## Отладка

- `?p=0.5` — поставить камеру в точку пути (0..1) без скролла
- `?focus=2` — автофокус на карточке услуги N
- `?flat=1` — принудительно плоская версия

## Запуск

Статика, сборки нет: `python3 -m http.server` и открыть `index.html`.
